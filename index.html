LuxGallery

Combined gallery and lightbox

Incredibly simple, all-in-one solution for web galleries

Instructions

  • Download LuxGallery
  • Upload the LuxGallery folder to the root of your website

About

LuxGallery requiers a web server with PHP 5.4+

I made this app because I wanted a simple, elegant photo gallery that was plug-and-play. Most importantly, I created LuxGallery to be accesible and and usable by everyone, not just developers.
